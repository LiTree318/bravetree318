tags: 
create-date: "{{date: YYYY-MM-DD-DDDD}}"
---


> [!info]+ ⚙️ 額外屬性
> ⌨ 輸入額外屬性
> ```	
> 	source:: 
> 	date::
> 	author::
> 	note::
> 	quotation:: 
> ```
> > [!example] 額外屬性說明
> > ```
> > // source 資料來源：網站／書籍／新聞等：
> > 	如果是網站：放上網站平台名稱（例如：CNN、Youtube）
> > 	如果是書籍中的文章：放上書籍名稱（例如：花蓮學研討會論文集）
> > 	如果是新聞：放上新聞媒體名稱（例如：自由時報、聯合報）
> > // date 放上資料的建置日期。
> > // author 放上資料的作者。
> > // note 放上我的評論或者註解。
> > // quotation 放上從資料中引用的內容。	
> > ```
